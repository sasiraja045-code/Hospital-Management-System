# 🏥 Hospital Management System

A structured SQL project built using Oracle SQL to manage core hospital operations including patient registration, doctor management, appointment scheduling, and medical records.

---

## Tech Stack

- **Database:** Oracle SQL
- **Tool:** Oracle Online Editor (OneCompiler)
- **Language:** SQL (DDL + DML)

---

## Tables

| Table | Description |
|---|---|
| `patients` | Stores patient personal details and blood group |
| `doctors` | Stores doctor details and specialization |
| `appointments` | Manages patient-doctor appointment scheduling |
| `medical_records` | Stores diagnosis, treatment and prescription details |

---

## Features

- Patient registration with blood group tracking
- Doctor management with specialization details
- Appointment booking with status tracking (Scheduled / Completed / Cancelled)
- Medical records with diagnosis and prescription details
- JOIN queries to fetch combined patient & doctor data
- Aggregate queries for doctor-wise appointment count

---

##  File

| File | Description |
|---|---|
| `Hospital.sql` | Complete SQL script — CREATE, INSERT, SELECT queries |

---

##  How to Run

1. Open [OneCompiler Oracle](https://onecompiler.com/oracle)
2. Copy the code from `Hospital.sql`
3. Paste and click **Run**
4. Output will display all tables and query results
